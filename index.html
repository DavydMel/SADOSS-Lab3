70-486: Developing ASP.NET MVC Web Applications перевіряє знання розробників в галузі розробки веб-додатків на платформі ASP.NET MVC, включаючи знання програмування мовою C# та використання баз даних SQL Server. Цей сертифікат може бути корисним для розробників, які працюють з платформою ASP.NET MVC та бажають підвищити свій професійний рівень.

98-361: Software Development Fundamentals перевіряє базові знання розробників у галузі програмування, включаючи знання основ програмування, алгоритмів та структур даних. Цей сертифікат може бути корисним для початківців розробників, які тільки починають свою кар'єру в галузі програмування та бажають отримати базові знання.
